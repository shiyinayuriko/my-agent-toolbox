# My Agent Toolbox

个人 AI agent 工具集合，按需安装。

## 安装方式

将以下内容发送给 AI 即可开始安装：

```
请阅读 https://raw.githubusercontent.com/shiyinayuriko/my-agent-toolbox/main/INSTALL.md 并引导我安装工具
```

## 工具列表

| 工具 | 版本 | 类型 | 说明 |
|------|------|------|------|
| [auto-review](./auto-review/) | 0.2.0 | plugin + agent | 操作安全性自动审核，安全操作自动放行 |
| [session-rename](./session-rename/) | 0.1.0 | plugin + command | `/session-rename` 命令，根据对话上下文重命名 session |
